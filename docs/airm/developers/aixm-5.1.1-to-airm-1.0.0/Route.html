Route

Route

A specified route designed for channelling the flow of traffic as necessary for the provision of air traffic services, from the end of the take-off and initial climb phase to the commencement of the approach and landing phase.

Properties Definition Type
annotation -
designatorNumber The number of the route designator. (unsignedint) NoNumberType
designatorPrefix A prefix for the route designator, indicating whether the route is upper ATS, reserved for helicopters or reserved for supersonic flights. (string) CodeRouteDesignatorPrefixType
designatorSecondLetter The letter of the route designator. Depending on the region, letters may have special significance with regard to the route usage limitations. (string) CodeRouteDesignatorLetterType
flightRule A code indicating the flight rules to be observed by aircraft. For example, VFR and/or IFR. (string) CodeFlightRuleType
internationalUse A code indicating if the route segment is international or domestic. (string) CodeRouteOriginType
locationDesignator A textual description of the area in which a designated route is situated. (string) TextDesignatorType
militaryTrainingType A code indicating the type of military training activity for which the route has been designed. (string) CodeMilitaryTrainingType
militaryUse A code indicating whether the route is for exclusive use by military aircraft. (string) CodeMilitaryStatusType
multipleIdentifier A suffix for the route designator, indicating alternates for homonymous routes (especially for military training routes). (string) CodeUpperAlphaType
name A free text name by which the route is known. (string) TextNameType
type Route classification. (string) CodeRouteType
userOrganisation Indicates the Organisation normally originating activity on a military training route. -
