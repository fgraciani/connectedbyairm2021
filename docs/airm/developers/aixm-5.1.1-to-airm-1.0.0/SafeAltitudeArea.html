SafeAltitudeArea

SafeAltitudeArea

A set of Minimum Sector Altitudes which completely describe an area centred on a significant point, usually contained within a sector of 46km (25NM) radius.

Properties Definition Type
annotation -
centrePoint The point used to build the Safe Altitude Area. -
location Indicates the localisation of the safe altitudes area at a specific airport -
safeAreaType Indicates the type of area, either MSA or ESA (string) CodeSafeAltitudeType
sector A subdivision of the Safe Altitude Area designed to provide the lowest altitude possible and maintain safety. (decimal) ValDistanceType
