AirspaceGeometryComponent

AirspaceGeometryComponent

The role of the component in the airspace geometry. If the geometry of an airspace is composed of single volume, then the attributes of this association class may be left empty.

Properties Definition Type
annotation -
operation An indicator of how the component participates in the aggregation, such as addition, subtraction or intersection. (string) CodeAirspaceAggregationType
operationSequence An indicator of the order of the component in an aggregation. (unsignedint) NoSequenceType
