OnlineContact

OnlineContact

On-line or Network information that can be used to contact the individual or organisation, including eMail address and web site page.

Properties Definition Type
eMail The address of the electronic mailbox of the responsible organisation or individual. (string) TextAddressType
linkage A contact's location (address) for on-line access using a Uniform Resource Locator address or similar addressing scheme such as http://www.statkart.no/isotc211 . (string) TextAddressType
network The official name of the telecommunication network on which a contact's online resource is located. (string) CodeTelecomNetworkType
protocol The connection protocol to be used for accessing an contact's online resource. (string) TextNameType
