Azimuth

Azimuth

A component of an MLS consisting of an SHF transmitter and associated equipment, radiating signals in a volume of airspace served by the MLS, thereby furnishing azimuth indications to aircraft approaching the runway or back azimuth indications to aircraft departing from the runway or performing a missed approach procedure.

Properties Definition Type
angleCoverLeft The value of the angle from the zero indication direction within which the azimuth indication is usable, and left of this direction from the azimuth antenna. (decimal) ValAngleType
angleCoverRight The value of the angle from the zero indication direction within which the azimuth indication is usable, and right of this direction from the azimuth antenna. (decimal) ValAngleType
angleProportionalLeft The value of the angle within which the azimuth indication is proportional to the deviation from the azimuth zero indication direction, and left of this direction from the azimuth antenna. (decimal) ValAngleType
angleProportionalRight The value of the angle within which the azimuth indication is proportional to the deviation from the azimuth zero indication direction, and right of this direction from the azimuth antenna. (decimal) ValAngleType
channel A code indicating the channel on which the Microwave Landing System is operating. (string) CodeMLSChannelType
magneticBearing The measured angle between the direction of the azimuth beam and Magnetic North at the azimuth antenna. (decimal) ValBearingType
trueBearing The measured angle between the azimuth beam and True North at the azimuth antenna. (decimal) ValBearingType
trueBearingAccuracy Accuracy of the measured angle between the azimuth beam and True North at the azimuth antenna (decimal) ValAngleType
type A code indicating if it is 'normal' or 'back' azimuth equipment. (string) CodeMLSAzimuthType
