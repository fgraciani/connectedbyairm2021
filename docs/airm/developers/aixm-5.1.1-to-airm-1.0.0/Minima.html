Minima

Minima

Vertical and horizontal visual minimums for a procedure segment conditions such as aircraft category, navigation equipment usage, procedure type, altimeter, etc.

Properties Definition Type
adjustmentINOP The inoperable equipment -
altitude The lowest altitude used in establishing compliance with obstacle clearance criteria. The lowest altitude an aircraft flies before making a decision to land or abort and take the missed approach. (Highest of the lower limit altitudes in controlling obstacles plus the altitude adjustments (operational factors)) (string) ValDistanceVerticalType
altitudeCode An indication of the meaning of the altitude value, such as obstacle clearance altitude or decision altitude. (string) CodeMinimumAltitudeType
altitudeReference A code indicating the reference for the altitude. (string) CodeVerticalReferenceType
annotation -
height The height value for the minimum (string) ValDistanceVerticalType
heightCode An indication of the meaning of the height value, such as obstacle clearance height or decision height. (string) CodeMinimumHeightType
heightReference The reference surface for the height value. (string) CodeHeightReferenceType
mandatoryRVR Indicates visibility measurement type. (string) CodeYesNoType
militaryHeight The height value for the minimum used by militaries, such as DoD in United States. (string) ValDistanceVerticalType
militaryVisibility The visibility value applicable to the military height. (decimal) ValDistanceType
radioHeight The height reported by a radio altimeter. Applies to CAT II. (string) ValDistanceVerticalType
remoteAltimeterMinima Indicates if this minima is the primary. (string) CodeYesNoType
visibility The minimum visibility needed to land at the airport when using the specified altitude and/or height. (decimal) ValDistanceType
