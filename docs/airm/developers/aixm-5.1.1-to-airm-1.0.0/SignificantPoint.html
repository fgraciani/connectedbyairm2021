SignificantPoint

SignificantPoint

A link class that allows selecting between a navaid system, a runway point, an airport reference point, an aiming point or a fix designated point. SignificantPoint accounts for a specified geographical location used to define an ATS route, the flight path of an aircraft or for other navigation/ATS purposes.

Properties Definition Type
aimingPoint The point at the centre of the TLOF. -
airportReferencePoint the point designated as the center of the airport. -
fixDesignatedPoint A selected point that may have references (distance and/or angle indications) -
navaidSystem A navaid used as significant point. -
position An unnamed point designated only with a latitude/longitude. -
runwayPoint A designated point on the runway centreline. -
