CircleSector

CircleSector

A group of properties that help describe a 3D pattern based on start angle, stop angle, inner distance, outer distance and upper/lower altitude.

Properties Definition Type
angleDirectionReference A code indicating a direction with regard to a reference point. (string) CodeDirectionReferenceType
angleType A code indicating the type of angle: magnetic, bearing, VOR radial. (string) CodeBearingType
annotation -
arcDirection Direction indicating clock-wise or counter-clockwise (string) CodeArcDirectionType
fromAngle Beginning of angle (decimal) ValBearingType
innerDistance Angle sector volume inner limit (decimal) ValDistanceType
lowerLimit The lowermost altitude or level that is included in the sector. (string) ValDistanceVerticalType
lowerLimitReference The reference surface used for the value of the lower limit. For example, Mean Sea Level, Ground, standard pressure, etc. (string) CodeVerticalReferenceType
outerDistance Angle sector volume outer limit (decimal) ValDistanceType
toAngle Ending of angle (decimal) ValBearingType
upperLimit The uppermost altitude or level that is included in the sector. (string) ValDistanceVerticalType
upperLimitReference The reference surface used for the value of the upper limit. For example, Mean Sea Level, Ground, standard pressure, etc.. (string) CodeVerticalReferenceType
