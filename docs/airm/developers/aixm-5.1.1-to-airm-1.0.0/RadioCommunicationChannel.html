RadioCommunicationChannel

RadioCommunicationChannel

A radio frequency band of sufficient width and associated identification data used for one- or two-way communication from or to a transmitter on the ground or in the air.

Properties Definition Type
annotation -
availability The operational status of the RadioCommunicationChannel. -
channel The identifier of the radio channel on which the communication takes place. (string) CodeCommunicationChannelType
emissionType A code indicating the type of emission, as defined at the 1979 ITU World Administrative Radio Conference. For example, A3E, etc.. (string) CodeRadioEmissionType
flightChecked Indicates if the communication channel has been flight checked. (string) CodeYesNoType
frequencyReception The value of the reception frequency. (decimal) ValFrequencyType
frequencyTransmission The value of the transmission frequency. (decimal) ValFrequencyType
location The location from where the radio are transmitted. -
logon A specified code used for data link, such as used for logon to an ATS unit. (string) TextDesignatorType
mode The type of communication channel. (string) CodeCommunicationModeType
rank A code indicating the role of the communication channel, in terms of primary, alternate, emergency, etc. (string) CodeFacilityRankingType
selectiveCall A code indicating if whether or not a frequency treats "selective call". (string) CodeYesNoType
trafficDirection Indicates the eventual directionality of the communication channel. (string) CodeCommunicationDirectionType
