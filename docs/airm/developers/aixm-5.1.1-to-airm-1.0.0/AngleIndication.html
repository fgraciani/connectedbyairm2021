AngleIndication

AngleIndication

An angular reference from a navaid or with reference to a designated point.

Properties Definition Type
angle The indication of a bearing (at a given point) by the measurement of the angle between the bearing and either True North or Magnetic North (this should appear explicitly or implicitly). The angle is measured clockwise from 0 degrees up to but not including 360 degrees. The value can also be a VOR radial. For example, Westward is expressed as 270. (decimal) ValBearingType
angleType A code indicating the direction of the zero bearing. For example, magnetic north or true north. (string) CodeBearingType
annotation -
cardinalDirection The angle expressed as compass direction. (string) CodeCardinalDirectionType
fix A selected point that may have references (distance and/or angle indications) -
indicationDirection Code indicating the if the angle is TO or FROM the Navaid or Designated Point that is used. (string) CodeDirectionReferenceType
minimumReceptionAltitude The lowest altitude above the mean sea level (MSL) where an aircraft can receive signal. (string) ValDistanceVerticalType
pointChoice A choice of a navaid, an airport reference point, a TLOF aiming Point, a fix , a lat/long indicated by point object, or a Rwy threshold which defines the facility makeup.. -
trueAngle The indication of a bearing (at a given point) by the measurement of the angle between the bearing and True North. The angle is measured clockwise from 0 degrees up to but not including 360 degrees. The value can also be a VOR radial. For example, Westward is expressed as 270. (decimal) ValBearingType
