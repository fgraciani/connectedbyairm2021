AerodromeRunwayState

AerodromeRunwayState

An aggregation of runway conditions typically reported together at an aerodrome, including the runway contamination, friction, and deposits.

Properties Definition Type
allRunways When true, indicates the reported conditions apply to all runways Boolean
cleared Indicates that reported runway has been cleared of meteorological deposits, such as snow. ICAO Annex 3 / WMO No. 49-2: CLRD" Section 4.8.1.5, Table A3-2 "State of the runway"" Boolean
contamination Proportion of runway surface that is contaminated - usually expressed as a percentage of the total runway area. WMO reporting procedures require that contamination is reported in the following categories: - less than 10%; - between 11% and 25%; - between 26% and 50%; and - more than 50%. See WMO No. 306: - WMO Code table 0519 - BUFR Code table 0 20 087 RunwayContamination
depositType The type of runway deposit, such as damp conditions, wet snow, or ice. WMO 306: Table 0919 RunwayDeposits
depthOfDeposit Depth of deposit on the surface of the runway. Measured depth of deposit shall be given in millimeters (mm"). See WMO No. 306 WMO Code table 1079." DistanceWithNilReason
estimatedSurfaceFrictionOrBrakingAction The estimated surface friction or braking action for the affected runway. When braking conditions are not reported and/or the runway is not operational estimatedSurfaceFrictionOrBrakingAction will not be reported. WMO 306: Table 0366 RunwayFrictionCoefficient
fromPreviousReport When true, indicates a new runway state report is not available in time for dissemination of the METAR message and the previous runway state report is being repeated Boolean
runway The runway to which the conditions apply. The runway may be missing in cases where all runways are closed due to snow RunwayDirection

Semantic Correspondence
AIRM Concept Definition
SurfaceContamination.wasCleared Indication that the surface was cleared.

contamination

AerodromeRunwayState.contamination

Proportion of runway surface that is contaminated - usually expressed as a percentage of the total runway area. WMO reporting procedures require that contamination is reported in the following categories: - less than 10%; - between 11% and 25%; - between 26% and 50%; and - more than 50%. See WMO No. 306: - WMO Code table 0519 - BUFR Code table 0 20 087

SurfaceContaminationLayer.contaminationExtent The proportion of a runway surface that is contaminated by a contaminant, as a percentage of the total runway area.
