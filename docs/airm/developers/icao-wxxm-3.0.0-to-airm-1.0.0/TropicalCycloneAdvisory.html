TropicalCycloneAdvisory

TropicalCycloneAdvisory

An advisory of tropical cyclone conditions of significance to aviation as represented in ICAO Annex 3 / WMO No. 49-2. Tropical cyclone advisories are issued by tropical cyclone advisory centres (TCACs)

Properties Definition Type
advisoryNumber Advisory number: year in full and message number (separate sequence for each cyclone) StringWithNilReason
forecast The forecast conditions of the tropical cyclone at given time intervals. TropicalCycloneForecastConditions
issueTime The time at which this advisory was issued TM_Instant
issuingTropicalCycloneAdvisoryCentre The issuing TCAC Unit
nextAdvisoryTime The time at which the next advisory is expected to be issued. Use attribute indeterminatePosition to element timePosition to indicate if the actual temporal position is before or after the specified value. When no subsequent advisory is expected to be issued it should be indicated by a missing expected advisory time with a nil reason of 'inapplicable' TM_Instant
observation The observed conditions of the tropical cyclone TropicalCycloneObservedConditions
remarks Remarks, as necessary. When the remark is nil, it should be indicated by a missing remarks with a nil reason of 'inapplicable' StringWithNilReason
tropicalCycloneName The tropical cyclone name. If the tropical cyclone is unnamed, 'NN' may be used TropicalCyclone
