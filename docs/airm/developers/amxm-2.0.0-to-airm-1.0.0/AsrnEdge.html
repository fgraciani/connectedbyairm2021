AsrnEdge

AsrnEdge

Definition: A directional or bi-directional connection between nodes having an attached geometry which can be used for the graphical depiction of the taxi route.

Properties Definition Type
curvatur Definition: Total course change along the edge -
direc Definition: Directionality of corresponding feature instance, which can be one-way or two-way -
edgederv Definition: Derivation method of edge geometry. Abstracted and partially abstracted edge geometries fulfill the functional aspect of connecting adjacent features, but may not be suitable for displaying a taxi route in all applications -
edgelen Definition: Length of the edge -
edgetype Definition: Type of ASRN edge -
geoline Definition: Line object -
idapron Definition: Name of apron -
idbase Definition: Name of corresponding feature instance -
node1ref Definition: The idnumber of the ASRN Node corresponding to the start point of the ASRN Edge geometry -
node2ref Definition: The idnumber of the ASRN Node corresponding to end point of the ASRN Edge geometry -
pcn Definition: A code expressing the bearing strength of a pavement for unrestricted operations</para><para> Encoding: expressed in the ICAO ACN-PCN Method as specified in ICAO Annex 14, Chapter 2.6</para><para> Example: PCN80/R/B/W/T -
restacft Definition: Usage restriction (prohibition) for specified aircraft type </para><para> Encoding: expressed according to ICAO Document 8643 - Aircraft Type Designators. </para><para> Example: "B744" for Boeing 747-400, "A388" for Airbus A380. </para><para> If more than one aircraft type is included in the usage restriction, the different types are divided by a period. </para><para> Example: "B744.A388" for a RunwayElement prohibiting both Boeing 747-400 and A380 aircraft. -
wingspan Definition: Maximum wingspan for this feature instance -
