You can download the AIRM UML model and the AIRM Rulebook.

Sparx Systems Enterprise Architect file 

The AIRM is maintained as a complex UML model using Sparx Systems Enterprise Architect.

Important Note: We recommend our advanced users to consider using the AIRM Viewer before downloading the .eap model.

An XMI 2.1 export of the AIRM UML model is also available.

The AIRM Viewer allows consulting the AIRM without the need of any download.

Rulebook

The AIRM Rulebook provides principles, rules and recommendation in order to facilitate the development and maintenance of the AIRM. 

The principles, rules and recommendations are intended to be used for modelling, consolidation, validation and verification, and quality check purposes.
