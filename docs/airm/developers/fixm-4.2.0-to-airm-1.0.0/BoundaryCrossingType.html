BoundaryCrossingType

fx:BoundaryCrossingType

Boundary Crossing contains estimate data conveyed between ATS Units during the process of Controller Coordination. [FIXM]

Properties Definition Type
altitudeInTransition Negotiated boundary crossing transition altitude. fx:AltitudeInTransitionType
clearedLevel The cleared altitude (flight level) at which the aircraft will cross the boundary crossing point if in level cruising flight or, if the aircraft is climbing or descending at the boundary crossing point, the cleared flight level to which it is proceeding. fb:FlightLevelOrAltitudeChoiceType
crossingPoint The point where the flight will cross an ATS facility boundary. fb:SignificantPointChoiceType
crossingTime The estimated time at which a flight will cross the associated boundary crossing point. fb:TimeType
extension An extension hook for attaching extension (non-core) classes. fb:BoundaryCrossingExtensionType
