fx:FlightType

The Flight element is an entry point to the FIXM model.

flightIdentification A container of flight identifying data. [FIXM] fx:FlightIdentificationType

radioCommunicationFailure Contains information about radio communication failure fx:RadioCommunicationFailureType
remarks Any other plain-language remarks when required by the appropriate ATS authority or deemed necessary, by the pilot-in-command for the provision of air traffic services. fb:CharacterStringType
routeTrajectoryGroup A logical grouping for all Route Trajectory Groups associated with the flight. [FIXM] fx:RouteTrajectoryGroupContainerType
specialHandling The reason for special handling of a flight by ATS. [adapted from ICAO Doc 4444, Appendix 2, ITEM 18 STS/] fx:SpecialHandlingReasonCodeListType
supplementaryData A container for supplementary data about the flight.
