TrajectoryPoint4DType

fx:TrajectoryPoint4DType

Identifies relevant information about a trajectory point including location, altitude, time, etc.

Properties Definition Type
altimeterSetting The barometric pressure reading used to adjust a pressure altimeter for variations in existing atmospheric pressure or to the standard altimeter setting (29.92). fb:PressureType
extension An extension hook for attaching extension (non-core) classes. fb:TrajectoryPoint4DExtensionType
level Altitude of the 4D point. fb:FlightLevelOrAltitudeChoiceType
metData In a predicted trajectory, the instantaneous temperature and wind vector used at the 4D Point for creating the 4D trajectory. fx:MeteorologicalDataType
pointProperty Describes any applicable properties of the trajectory point. May include multiple properties per point. fx:TrajectoryPointPropertyType
position The geographical location of the 4D point. fb:GeographicalPositionType
predictedAirspeed The predicted indicated airspeed or Mach at the trajectory point. fb:IndicatedAirspeedType
predictedGroundspeed The predicted ground speed at the Trajectory Point in knots or kilometers per hour. fb:GroundSpeedType
time 4D Point time expressed as either absolute or relative time. fx:Point4DTimeChoiceType
verticalRange Vertical Range representing block altitude clearances fb:VerticalRangeType
