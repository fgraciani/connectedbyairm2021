CruisingLevelChangeType

fx:CruisingLevelChangeType

Describes a new planned altitude or level at the specified point at which the change is planned to commence. (May also describe a point at which the altitude or level change is planned to be attained.)

Properties Definition Type
activation Qualifies the position associated with the level constraint. fx:ActivationType
extension An extension hook for attaching extension (non-core) classes. fb:CruisingLevelChangeExtensionType
level A new planned altitude or level at the specified point at which the change is planned to commence fb:FlightLevelOrAltitudeChoiceType
