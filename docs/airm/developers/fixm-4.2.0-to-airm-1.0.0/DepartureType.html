DepartureType

fx:DepartureType

Groups information pertaining to the flight's departure.

Properties Definition Type
actualTimeOfDeparture The actual time of departure from the aerodrome. [ICAO Doc 4444, Appendix 3, Field Type 13] fb:TimeType
aerodrome The aerodrome from which the flight departs. [FIXM] fb:AerodromeReferenceType
aerodromePrevious Specifies the previous departure aerodrome value when a change is made. fb:AerodromeReferenceType

airportSlotIdentification The identifier of the departure airport slot allocated to the flight. [FIXM] fb:AirportSlotIdentificationType
estimatedOffBlockTime The estimated time at which the aircraft will commence movement associated with departure. [ICAO Doc 4444] fb:TimeType
estimatedOffBlockTimePrevious Specifies the previous estimated off block time value when a change is made. fb:TimeType
extension An extension hook for attaching extension (non-core) classes. fb:DepartureExtensionType
runwayDirection The runway direction used for taking off. [FIXM] fb:RunwayDirectionDesignatorType
takeoffAlternateAerodrome An alternate aerodrome at which an aircraft can land should this become necessary shortly after take-off and it is not possible to use the aerodrome of departure. [ICAO Doc 4444] fb:AerodromeReferenceType
