OccupancyTrafficMonitoringValue European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:DemandAndCapacityBalancing:OccupancyTrafficMonitoringValue

A set of values (peak, sustain, overload duration, duration of counting) dedicated to monitor the instantaneous density of aircraft in a sector, as represented in occupancy counts.

Source: SESAR
Abbreviations: OTMV
Supplements: TemporalEnabledEntity

Properties:

Properties Definition Type
crossingOccurrences Number of crossing occurrences of the sustained threshold within elapsed, before the Occupancy Traffic Monitoring Value (OTMV) triggers an alert.

Integer

monitoredTFV The traffic volume for which the occupancy traffic monitoring value is defined. TrafficVolume
sustainedThreshold Acceptable number of flights that could be handled by a sector under specific circumstance, in particular if the duration of the overload is not too long.

Integer

peakThreshold Maximum number of flight that could be handled by a sector.

Integer

overloadDuration Maximum duration beyond which a potential hotspot should be considered in case the occupancy count exceeds the sustain threshold.

TM_Duration

elapsed Duration of the time window on which the crossing occurrences are counted.

TM_Duration
