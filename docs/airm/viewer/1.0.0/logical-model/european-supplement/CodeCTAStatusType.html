CodeCTAStatusType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeCTAStatusType

A code indicating the possible statuses of the CTA negotiation process.

Source: SESAR

Properties:

Properties Definition Type
CTA_ALLOCATED A status code indicating that the system has generated a CTA for the flight.

enum value

I4D_NOT_ELIGIBLE A status code indicating that the flight cannot be handled using i4D.

enum value

I4D_LOGGED_ON A status code indicating that the flight in question is logged on to air/ground communications with the i4D enabled equipment (2nd precondition of i4D operations).

enum value

I4D_EQUIPPED A status code indicating that the CTA process has proceed to the stage that the aircraft executing the flight is suitably equipped to be subjected to i4D procedures (1st precondition of i4D operations).

enum value

I4D_ELIGIBLE A status code indicating that the flight is eligible for i4D operations (this requires not only equipment but also certain flight conditions to be met).

enum value

CTA_UNSUCCESSFUL A status code indicating that the CTA process has failed, i.e. no CTA constraint could be established.

enum value

CTA_SUITABLE A status code indicating that the flight deck has agreed to the proposed CTA/CTO.

enum value

CTA_PROPOSED A status code indicating that the i4d process had been initiated.

enum value
