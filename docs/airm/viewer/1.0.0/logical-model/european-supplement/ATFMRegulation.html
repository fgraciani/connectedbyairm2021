ATFMRegulation European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:DemandAndCapacityBalancing:ATFMRegulation

An ATFCM measure implemented by means of a departure slot in order to balance traffic demand against available ATC capacity.

Source: EUROCONTROL CFMU_B2B_FlowServices
Synonyms: SlotAllocationMeasure
Parent concept: : ATFMMeasure

Properties:

Properties Definition Type
designator Unique and immutable designator of the ATFM regulation within an AIRAC cycle.

CharacterString

slotAllocation ATFM departure slot resulting from the ATFM regulation. ATFMDepartureSlot
protectedLocation Reference location that the regulation is meant to protect, when the protected reference location is not the reference location at which the regulation applies. ReferenceLocation
delayTrafficVolumeSet The traffic volume set to which the delay of this regulation is associated. TrafficVolumeSet
slotAllocationList The slot allocation lists which apply to the regulation. SlotAllocationList
linkedRegulation ATFM regulation which is linked to this regulation. ATFMRegulation
regulationRate The coordinated number of flights that can be accommodated in a certain time period. RegulationRate
suplementaryRate Supplementary rate information expressed by a regulation for a specific supplementary period, expressed as number of slots per hour. SupplementaryRate
regulationCauseLocationCategory The location category for an ATFM regulation. CodeRegulationCauseLocationCategoryType
regulationCause A structured code indicating the reason for implementing an ATFM Regulation. CodeRegulationCauseType
noDelayWindow Time window duration around the Estimated Time Over (ETO) within which the regulation does not delay a flight. In certain cases, the window is applied around the minimum Controlled Time Over (CTO).

TM_Duration

isTFVActivationUpdateRequired Indication that a traffic volume activation will be achieved according to the applicability period of the regulation.

Boolean

isFlightConfirmationMandatory Indication whether the flight must be confirmed in this regulation before trying to find a slot for this flight in the associated constraint period.

Boolean

isCapacityUpdateRequired Indicates that a capacity update will be automatically applied according to the rates of the regulation.

Boolean

iataDelayCode The IATA delay code of the regulations due to this regulation cause.

Integer
