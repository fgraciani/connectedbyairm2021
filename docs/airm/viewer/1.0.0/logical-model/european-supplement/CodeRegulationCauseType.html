CodeRegulationCauseType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeRegulationCauseType

A structured code indicating the reason for implementing an ATFM Regulation.

Source: EUROCONTROL CFMU-B2B
Synonyms: CodeRegulationReasonType
