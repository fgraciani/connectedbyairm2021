CodeTacticalReroutingStatusType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeTacticalReroutingStatusType

If a flight was rerouted, describes the current rerouting state.

Source: EUROCONTROL CFMU-B2B

Properties:

Properties Definition Type
EXECUTED Rerouting has been executed

enum value

TIMED_OUT No FPL / CHG received on time

enum value

REVOKED A booking was created for the proposal, but a manual operation that changed the regulation (typically deep rectify) has deleted the booking before the attempt to accept the proposal

enum value

REJECTED Rerouting proposal has been rejected

enum value

PRODUCED There is a valid rerouting going on, waiting to be realised by either an FPL or a CHG

enum value

NO_MATCH Message received did not match the proposal; rerouting has been invalidated

enum value
