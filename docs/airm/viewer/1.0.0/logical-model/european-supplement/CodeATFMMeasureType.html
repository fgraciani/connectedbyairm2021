CodeATFMMeasureType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeATFMMeasureType

Type of the ATFM Measure.

Source: EUROCONTROL CFMU-B2B

Properties:

Properties Definition Type
AIRBORNE_HORIZONTAL_REROUTING A tactical ATFM measure where the aircraft is rerouted while airborne.

enum value

TERMINAL_PROCEDURE_CHANGE A tactical ATFM measure which requires the aircraft to use a different terminal procedure.

enum value

TAKE_OFF_NOT_BEFORE A tactical ATFM measure where the aircraft shall take-off not before the required time.

enum value

TAKE_OFF_NOT_AFTER A tactical ATFM measure where the aircraft shall take-off not after the required time.

enum value

MINIMUM_DEPARTURE_INTERVALS A tactical ATFM measure carried out when ATC sets a departure flow rate of, for example, 3 minutes between successive departures.

enum value

MILES_MINUTES_IN_TRAIL A tactical ATFM measure expressed as the number of minutes required between successive aircraft or a tactical ATFM measure expressed as the number of miles required between aircraft (in addition to the minimum longitudinal requirements) to meet a specific criterion which may be separation, airport, fix, altitude, sector or route specific.

enum value

GROUND_LEVEL_CAP A tactical ATFM measure applicable to not yet airborne flight(s) which limits the flight level at which an aircraft can fly.

enum value

GROUND_DELAY A strategic, pre-tactical, or tactical ATFM measure where aircraft are held on the ground in order to manage capacity and demand in a specific volume of airspace or at a specific airport.

enum value

AIRBORNE_LEVEL_CAP A tactical ATFM measure applicable to airborne flight(s) which limits the flight level at which an aircraft can fly.

enum value
