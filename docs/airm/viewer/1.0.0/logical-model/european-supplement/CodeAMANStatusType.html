CodeAMANStatusType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeAMANStatusType

A code indicating the status of eligibility of a flight for the AMAN processing.

Source: SESAR

Properties:

Properties Definition Type
CTA_ISSUED A status code indicating that the controller has communicated a CTA constraint.

enum value

TTA_ISSUED A status code indicating that the controller has issued a target time to the flight, typically resulting in the communication of an advisory.

enum value

TIME_FROZEN The amanTarget LandingTime (ATLDT) computed by the AMAN will no longer be automatically optimized.

enum value

SEQUENCED_UNSTABLE The flight is preliminary sequenced but the information is not stable enough to act on the flight.

enum value

SEQUENCED_STABLE The flight is sequenced and the information is stable enough to act on the flight.

enum value

SEQUENCED_FROZEN The position of the flight in the sequence will no longer be altered automatically by the AMAN optimization algorithm (it may change for operational reasons such as go-around of another a/c).

enum value

MANUAL_INTERVENTION The Arrival Sequence proposal has been overridden or a constraint/sequence position has been manually fixed for the flight(s) in question.

enum value

IS_ON_FINAL (added to provide A-CDM milestones using AMAN interface)

enum value

HAS_LANDED (added to provide A-CDM milestones using AMAN interface)

enum value

EXEMPTED A status code indicating that the flight is exempted from AMAN handling (e.g. due declared emergency).

enum value

ENABLED A status code indicating that AMAN sequencing is enabled.

enum value

DISABLED A status code indicating that AMAN sequencing is disabled.

enum value

DESEQUENCED A flight is temporarily removed from the sequence from operational reasons.

enum value
