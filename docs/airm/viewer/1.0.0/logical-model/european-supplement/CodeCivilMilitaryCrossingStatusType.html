CodeCivilMilitaryCrossingStatusType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeCivilMilitaryCrossingStatusType

The status of the crossing.

Source: FOIPS

Properties:

Properties Definition Type
ACCEPTED The crossing conditions (either first request or subsequent request or alternatively proposed) are accepted by the other unit.

enum value

REVISED_COUNTERED The other unit counter-proposed alternative crossing conditions in response to a request to change the previously agreed conditions (revised"), instead of accepting or rejecting the conditions proposed to it."

enum value

REVISED A unit requested some changes to previously agreed crossing conditions. These new conditions have still to be agreed.

enum value

REQUESTED The crossing is requested formally to the downstream unit (first request).

enum value

REJECTED The crossing conditions are rejected by the other unit.

enum value

NOTIFIED The crossing is notified to the downstream unit.

enum value

NO_CROSSING The crossing is not yet notified to the downstream unit.

enum value

COUNTERED The other unit counter-proposes alternative crossing conditions in reply to the first crossing request (before any conditions have ever been agreed between the two units).

enum value

CANCELLED The crossing is cancelled.

enum value
