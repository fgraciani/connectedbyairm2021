CodeReasonCancellationType European Supplement

urn:aero:ses:eurocontrol:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:Codelists:CodeReasonCancellationType

A coded list of values indicating the reason for abandoning the use of an airspace by a planned military mission.

Source: SESAR
