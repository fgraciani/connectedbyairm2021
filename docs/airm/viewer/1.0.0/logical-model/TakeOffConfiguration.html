TakeOffConfiguration

urn:aero:airm:1.0.0:LogicalModel:Subjects:Aircraft:TakeOffConfiguration

Aircraft configuration for a given flight. It is the configuration at take-off"."

Source: SESAR
Parent concept: FlightConfiguration

Properties:

Properties Definition Type
cargoWeight The weight of the cargo of the flight. ValMassType
constraint A constraint on the flight configuration caused by the operator's configuration. OperatorConfiguration
flight The flight for which the configuration applies. Flight

takeOffWeight The weight of the aircraft at the beginning of the take-off phase. ValMassType
numberOfTrafficUnits Number of traffic units for the flight. One traffic unit is equivalent to one passenger or 100Kg air freight or 10 kg air mail.

Integer

personsOnBoard The total number of persons (passengers and crew) on board.

Integer
