CodeRouteTrajectoryPointUsageType

urn:aero:airm:1.0.0:LogicalModel:Subjects:Flight:Codelists:CodeRouteTrajectoryPointUsageType

A code indicating the usage of the route/trajectory point.

Source: SESAR

Properties:
