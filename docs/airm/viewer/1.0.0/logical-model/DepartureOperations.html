DepartureOperations

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:AerodromeOperations:DepartureOperations

The set of flight operations following procedures associated with the departure aerodrome.

Source: SESAR
Parent concept: TemporalEnabledEntity

Properties:

Properties Definition Type

sid The standard instrument departure the aircraft may use after taking off. StandardInstrumentDeparture

offBlock Off-Block event linked to the departure. OffBlock

boarding The boarding of the aircraft during the departure operations. Boarding
deicingProcedure The procedure describing the progress in de-icing process. Deicing
