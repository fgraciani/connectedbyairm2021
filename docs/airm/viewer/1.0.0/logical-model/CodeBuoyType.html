CodeBuoyType

Code indicating the type of Buoy

Source: EUROCONTROL-FAA AIXM v.5.1

Properties:

Properties Definition Type

WHITE No colour is stated on the chart

enum value

enum value

enum value

enum value

enum value

enum value

enum value
