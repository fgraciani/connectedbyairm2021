CodeVisualDockingGuidanceType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeVisualDockingGuidanceType

A coded list of values that indicate types of systems used at aircraft stands in order to help the pilot align and position the aircraft.

Source: EUROCONTROL-FAA AIXM v.5.1

Properties:

Properties Definition Type
A_VDGS Advanced Visual Docking Guidance System that features electronic displays which perform the functions of an AGNIS/PAPA installation, although with much greater accuracy. They may also provide collision avoidance from static objects.

enum value

SAFE_GATE A type of A-VDGS that uses an aircraft symbol on a display indicating the relative position along the centreline.

enum value

SAFE_DOC Safe dock allows aircraft to park up to an accuracy of 10 cm using lasers to attain the aircraft's position.

enum value

PAPA Parallax Aircraft Parking Aid consisting of a large grey/black box that uses the effect of perspective in order to indicate the relative position of the aircraft along the centreline.

enum value

APIS Aircraft Positioning and Information System that shows azimuth information using a series of parallel and black bars which kink at their midpoint.

enum value

AGNIS_STOP AGNIS system complemented with a simple stop light, on the side of the AGNIS display.

enum value

AGNIS_PAPA Combined AGNIS and PAPA device.

enum value

AGNIS Azimuth Guidance for Nose-In Stand, providing centreline guidance only by using two coloured lights mounted side by side.

enum value
