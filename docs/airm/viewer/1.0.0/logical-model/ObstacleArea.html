ObstacleArea

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Obstacle:ObstacleArea

An area used for the purpose of collecting obstacle data, as necessary for air navigation systems or functions.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: GeoTemporalEnabledEntity

Properties:

Properties Definition Type
obstructionIdentificationSurfaceCondition An indicator of the type of obstacle assessment surface that obstructing area represents. CodeObstacleAssessmentSurfaceType
ownerAerodrome The aerodrome for which the obstacle area has been defined. Aerodrome
ownerRunway The ruwnay for which the obstacle area has been defined. RunwayDirection
obstacle Obstale within the obstacle area. VerticalStructure
ownerOrganisation The organisation for which the obstacle area has been defined. Organisation
type The type of obstacle coverage area. CodeObstacleAreaType
