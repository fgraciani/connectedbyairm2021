SafeAltitudeArea

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:RouteAndProcedure:SafeAltitudeArea

A set of Minimum Sector Altitudes which completely describe an area centred on a significant point, usually contained within a sector of 46km (25NM) radius.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: TemporalEnabledEntity

Properties:

Properties Definition Type
safeAreaType The type of area. CodeSafeAltitudeType
centrePoint The point used to build the safe altitude area. InfrastructurePoint
location The localisation of the safe altitudes area at a specific aerodrome. Aerodrome
sector A subdivision of the safe altitude area designed to provide the lowest altitude possible and maintain safety. SafeAltitudeAreaSector
