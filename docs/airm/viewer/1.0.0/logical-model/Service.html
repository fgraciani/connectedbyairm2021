Service

urn:aero:airm:1.0.0:LogicalModel:Subjects:Stakeholders:BusinessService:Service

The provision of tangible goods, information, instructions or guidance.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: GeoTemporalEnabledEntity
Children: AirportGroundService; AirTrafficManagementService; InformationService; SearchAndRescueService; TrafficSeparationService;

Properties:

Properties Definition Type
flightOperations The destination (arriving, departing, etc.) of the flights for which the information is provided. CodeFlightServiceAvailabilityType
radioCommunication The radio frequency on which the service is provided. RadioCommunicationChannel
groundCommunication The point of contact on the ground for the service. ContactInformation
serviceProvider The unit that provides the service. Unit
availability The operational status of the service. ServiceOperationalStatus
callSign The callsign of the unit providing the service. ServiceProviderCallsign
rank The order of priority of the service, such as primary" or "alternate". " CodeFacilityRankingType
name A free text name by which the service is identified.

CharacterString

isCompliantICAO A code indicating whether the service is compliant with the ICAO standards and recommended practices specified for this type of service.

Boolean
