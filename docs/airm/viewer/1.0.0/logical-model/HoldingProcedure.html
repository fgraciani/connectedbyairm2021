HoldingProcedure

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:RouteAndProcedure:HoldingProcedure

A predetermined manoeuvre which keeps an aircraft within a specified airspace while awaiting further clearance.

Source: ICAO Annex 4
Synonyms: HoldingPattern
Parent concept: GeoTemporalEnabledEntity

Properties:

Properties Definition Type
outboundLegEndPoint The second waypoint of a two point holding, used to define the end of the outbound leg. InfrastructurePoint
holdingFix A geographical location that serves as a reference for a holding procedure. InfrastructurePoint

outboundLegEndTime The duration of the holding pattern outbound leg. TM_Duration
speedLimit Restricted speed for containment in a smaller pattern during holding operations. ValSpeedType
outboundCourse The outbound course (for example, true track, magnetic track, heading, VOR radial, true bearing, magnetic bearing) from the significant point on which holding is based. Angle
inboundCourse The radial, course or bearing of the inbound leg as it pertains to a holding procedure. Angle
instruction Operational instructions that must be observed when flying the holding procedure. CharacterString
isNonStandardHolding An indication that the holding pattern is non-standard. Boolean
lowerLimit The minimum vertical distance, measured from a specified reference, allowed for all aircraft categories, at all defined speeds, operating in a holding procedure. ValVerticalDistanceType
outboundCourseType The type of outbound course to be observed. CodeCourseType
upperLimit The maximum vertical distance, measured from a specified reference, allowed for all aircraft categories, at all defined speeds, operating in a holding procedure. ValVerticalDistanceType
type An indicator of the type of holding procedure. CodeHoldingUsageType
turnDirection An indicator of the turn direction in an aerial manoeuvre performed by aircraft during holding operations. CodeDirectionTurnType
