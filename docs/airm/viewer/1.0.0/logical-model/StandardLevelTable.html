StandardLevelTable

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:Airspace:StandardLevelTable

A table of consecutive cruising levels described under vertical separation criteria limited by an upper and lower level and used by General Air Traffic.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: TemporalEnabledEntity

Properties:

Properties Definition Type
isStandardICAO An indication whether the table is according to the ICAO SARPS.

Boolean
