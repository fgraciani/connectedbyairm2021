State

urn:aero:airm:1.0.0:LogicalModel:Subjects:Stakeholders:Stakeholder:State

An internationally recognized geographic entity that provides air traffic service.

Source: ICAO Doc 9881
Parent concept: Organisation

Properties:

Properties Definition Type
iataCountryCode Country codes as defined by IATA and ISO. CodeIATACountryCodeType
airspace Airspace located within the state borders. Airspace
city A city belonging to the state. City
border The border of the state. GeoBorder
role Role of a Country with respect to Nas. CodeNasRoleType
isEcacState Indicates if the Country is one of the ECAC states.

Boolean
