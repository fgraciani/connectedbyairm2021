SurveillanceRadar

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:SurveillanceInfrastructure:SurveillanceRadar

Radar equipment used to determine the position of an aircraft in range and azimuth.

Source: ICAOICAO Annex 1ICAO Annex 10
Parent concept: RadarEquipment

Properties:

Properties Definition Type
verticalCoverageAzimuth Azimuth for which the radar provides vertical coverage. Angle

verticalCoverageDistance Distance for which the radar provides vertical coverage. ValDistanceType
groundStation The ground station where the radar scope is utilised. Unit
tiltAngle Angle at which the antenna is tilted. Angle
isAntennaTiltFixed Indication whether the antenna tilt is fixed. Boolean
