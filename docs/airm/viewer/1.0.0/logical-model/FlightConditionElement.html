FlightConditionElement

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirTrafficOperations:DemandAndCapacityBalancing:FlightConditionElement

An operand in a flight condition combination.

Source: EUROCONTROL-FAA AIXM v.5.1
