RadioNavigationAid

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:NavigationInfrastructure:RadioNavigationAid

Any electronic system which provides information to be used by the pilot or aircraft navigation systems for position determination or flight path guidance.

Source: ICAO Doc 9294
Synonyms: RadioNavaidEquipment
Parent concept: GeoTemporalEnabledEntity
Children: DirectionFinder; DME; InstrumentLandingSystem; MarkerBeacon; MicrowaveLandingSystem; NDB; SimplifiedDirectionalFacility; TACAN; VOR;
