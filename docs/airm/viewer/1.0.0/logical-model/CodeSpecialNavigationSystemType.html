CodeSpecialNavigationSystemType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeSpecialNavigationSystemType

A code indicating a type of special navigation system. For example, LORAN, DECCA, GNSS, etc.

Source: EUROCONTROL-FAA AIXM v.5.1
