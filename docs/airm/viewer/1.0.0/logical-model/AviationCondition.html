AviationCondition

urn:aero:airm:1.0.0:LogicalModel:Subjects:Meteorology:AviationCondition

Weather observations and or forecast for a specific aeronautical location or airspace volume.

Source: EUROCONTROL-FAA WXXM v.1.1
Parent concept: WeatherCondition
Children: AerodromeCondition; AirspaceCondition; RouteCondition; RunwayCondition;
