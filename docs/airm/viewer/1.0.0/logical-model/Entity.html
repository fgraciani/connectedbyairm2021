Entity

urn:aero:airm:1.0.0:LogicalModel:Abstract:Entity

A definition (type) of an data (ATM) item of interest that is implementation independent and it is subject to constraints.

Source: SESAR
Supplements: Entity

Properties:

Properties Definition Type
metadata Metadata about the Entity. MD_Metadata
identifier Special unique identifier permanently assigned to an entity. IdentifierType
contextUsage The intended purpose of the data, whether that data is intended for publication, future development, research and development, or future predictions. CharacterString
lifecycleStageStatus The lifecycle stage of data to enable controls, such as allowing or prohibiting changes to data that is in a specific status, or making data available to other systems when it is in a certain status. CharacterString
annotation A general text note for the entity or for one of its properties. Note
