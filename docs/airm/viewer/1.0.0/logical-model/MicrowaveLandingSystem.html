MicrowaveLandingSystem

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:NavigationInfrastructure:MicrowaveLandingSystem

A precision approach and landing guidance system operating in the microwave spectrum, which provides position information and various ground-to-air data.

Source: EUROCONTROL ATM Lexicon
Abbreviations: MLS
Parent concept: RadioNavigationAid

Properties:

Properties Definition Type

mlsElevation The MLS Elevation component of the MLS. MLSElevation
