RadioFrequencyArea

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:NavigationInfrastructure:RadioFrequencyArea

A limitation, such as coverage, usability, etc., of a radio navigation aid or service frequency.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: GeoTemporalEnabledEntity

Properties:

Properties Definition Type
angleScallop The value of the scalloping angle, which may affect the radial reading of a VOR or TACAN in plus or minus.

Angle

sector A non graphical definition describing a portion of a circle associated with the radio frequency area. CircleSector
par The precision approach radar affected by the signal limitation. PrecisionApproachRadar
specialNavigationStation A land based station of a special navigation system. SpecialNavigationStation
ssr The secondary surveillance radar affected by the signal limitation. SecondarySurveillanceRadar
radioNavaidEquipment The component of a system used for air navigation. RadioNavigationAid
frequency A frequency in the range within which radio waves may be transmitted. RadioCommunicationChannel
type A code indicating the type of radio navigation aid limitation. CodeRadioFrequencyAreaType
signalType The type of signal that is affected by the usage limitation. CodeRadioSignalType
