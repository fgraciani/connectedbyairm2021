CodeSurfaceWeatherObservationSystem

urn:aero:airm:1.0.0:LogicalModel:Subjects:Meteorology:Codelists:CodeSurfaceWeatherObservationSystem

The types of surface weather observation systems.

Source: FAA (Federal Aviation Administration)

Properties:

Properties Definition Type
AUTOMATED_SURFACE_OBSERVING_SYSTEM Automated sensor suites that are designed to serve meteorological and aviation observing needs.

enum value

STANDALONE_WEATHER_SENSOR A standalone weather sensor.

enum value

RUNWAY_VISUAL_RANGE_SYSTEM System for measuring runway visual range.

enum value

LOW_LEVEL_WIND_SHEAR_ALERT_SYSTEM A Low level wind shear alert system measures average surface wind speed and direction using a network of remote sensor stations, situated near runways and along approach or departure corridors at an airport.

enum value

JUNEAU_AIRPORT_WIND_SYSTEM AWS generates turbulence alerts every minute for hazardous areas in the vicinity of Juneau airport.

enum value

AUTOMATED_WEATHER_SENSOR_SYSTEM An advanced AWOS system. It utilizes the complex National Weather Service ASOS algorithms for data processing and distribution.

enum value

AUTOMATED_WEATHER_OBSERVING_SYSTEM Automated Weather Observing System (AWOS) is a fully configurable airport weather system that provides continuous, real time information and reports on airport weather conditions. AWOS stations are mostly operated, maintained and controlled by aviation service providers.

enum value
