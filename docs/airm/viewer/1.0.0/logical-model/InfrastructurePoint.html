InfrastructurePoint

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:InfrastructurePoint:InfrastructurePoint

A point used in describing the base or airspace infrastructure.

Source: ICAO Information Management Panel (IMP)
Parent concept: SignificantPoint

Properties:
