Properties Definition Type
AIRBORNE_SEGMENT The airborne segment of the trajectory describes the anticipated 4D path of the aircraft using data elements and resolution necessary to deliver a level of fidelity commensurate with requirements for each portion of the trajectory.

enum value

enum value

TAXI_PATH Part of the agreed 4D trajectory for both Departure and Arrival surface segments.

enum value

enum value

DEPARTURE_SURFACE_SEGMENT This segment describes the elements of the overall trajectory from the departure gate up to and including the departure runway.

enum value

ARRIVAL_SURFACE_SEGMENT_ELEMENT A portion of the arrival surface segment.

enum value

enum value

enum value
