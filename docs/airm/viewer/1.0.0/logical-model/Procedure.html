Procedure

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:RouteAndProcedure:Procedure

A series of predetermined manoeuvres with specified protection from obstacles.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: TemporalEnabledEntity
Children: InstrumentApproachProcedure; StandardInstrumentArrival; StandardInstrumentDeparture;
