CodeRunwayProtectionAreaType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeRunwayProtectionAreaType

A code indicating the type of protection area for a runway For example, clearway, obstacle free zone, ILS protection area, etc..

Source: EUROCONTROL-FAA AIXM v.5.1

Properties:

Properties Definition Type
CLEARWAY A defined rectangular area on the ground or water under the control of the appropriate authority, selected or prepared as a suitable area over which an aeroplane may make a portion of its initial climb to a specified height.

enum value

VGSI_PROTECTION_AREA Visual Glide Slope Indicator (VGSI) protection area. Protects VGSI signal coverage by forbidding objects in the area.

enum value

STOPWAY A defined rectangular area on the ground at the end of take-off run available prepared as a suitable area in which an aircraft can be stopped in the case of an abandoned take-off.

enum value

RESA An area symmetrical about the extended runway centre line and adjacent to the end of the strip primarily intended to reduce the risk of damage to an aeroplane undershooting or overrunning the runway.

enum value

POFZ Precision Obstacle Free Zone

enum value

OFZ Obstacle free zone or surface

enum value

IOFZ Inner Obstacle free zone or surface

enum value

ILS_PROTECTION_AREA ILS protection area. Protects ILS signal distortion by forbidding large objects in the area.

enum value
