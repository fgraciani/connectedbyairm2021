RouteTrajectoryPoint

urn:aero:airm:1.0.0:LogicalModel:Subjects:Flight:Movement:RouteTrajectoryPoint

A significant point used in defining the flight path of an aircraft.

Source: ICAO Information Management Panel (IMP)
Parent concept: SignificantPoint

Properties:
