CodeServiceInformationType

urn:aero:airm:1.0.0:LogicalModel:Subjects:Stakeholders:Codelists:CodeServiceInformationType

A list of codes used to designate information provision services.

Source: EUROCONTROL-FAA AIXM v.5.1
