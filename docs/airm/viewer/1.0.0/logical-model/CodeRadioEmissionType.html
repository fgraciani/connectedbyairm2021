CodeRadioEmissionType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeRadioEmissionType

A code indicating the type of emission, as defined at the 1979 ITU World Administrative Radio Conference. For example, A3E, NONA2a, G1D.

Source: EUROCONTROL-FAA AIXM v.5.1
