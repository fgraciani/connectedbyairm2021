RouteAvailability

urn:aero:airm:1.0.0:LogicalModel:Subjects:AirspaceInfrastructure:RouteAndProcedure:RouteAvailability

The operational availability of a route.

Source: EUROCONTROL-FAA AIXM v.5.1
Parent concept: ObjectWithSchedule

Properties:

Properties Definition Type
cardinalDirection The direction for which the usage is specified, expressed as compass angle. CodeCompassPointType
hostAirspace The Flight Information Region where the route opening/closure is located. Airspace
level The airspace layer referenced by the route usage. AirspaceLayer
status The availability status of the route, in the direction indicated by the direction and/or cardinalDirection. CodeRouteAvailabilityType
direction The direction in which the route segment is considered when a specific usage is described. CodeDirectionType
conditionalRouteType The category of the conditional route according to the Flexible Use of Airspace rules. CodeConditionalRouteType
