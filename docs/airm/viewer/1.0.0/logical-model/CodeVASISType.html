CodeVASISType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeVASISType

A code indicating a type of visual approach slope indicator system. For example, VASIS, AVASIS, 3-BAR VASIS, PAPI, APAPI, etc..

Source: EUROCONTROL-FAA AIXM v.5.1
