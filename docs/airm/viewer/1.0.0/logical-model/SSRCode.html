SSRCode

urn:aero:airm:1.0.0:LogicalModel:Subjects:Flight:FlightIdentifier:SSRCode

The number assigned to a particular multiple pulse reply signal transmitted by a transponder in Mode A or Mode C.

Source: ICAO Doc 4444
Parent concept: Entity

Properties:

Properties Definition Type
code Four numerics giving the SSR code assigned to the aircraft by ATS.

CharacterString

modeSIdentifier The ICAO aircraft address used in Mode S. AircraftAddress
status The status of the SSR code. CodeSSRCodeStatusType
modeSType Type of mode S CodeModeSType
mode The surveillance system used for the SSR code. CodeSSRModeType
codeCategory The category of code according to the type of flight. CodeSSRCategoryType
