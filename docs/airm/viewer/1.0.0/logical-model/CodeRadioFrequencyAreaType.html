CodeRadioFrequencyAreaType

urn:aero:airm:1.0.0:LogicalModel:Subjects:BaseInfrastructure:Codelists:CodeRadioFrequencyAreaType

A code indicating the type of radio navigation aid limitation.

Source: EUROCONTROL-FAA AIXM v.5.1

Properties:

Properties Definition Type
COV Actual coverage range (flight checked). Equivalent to Operational Service Volume.

UNREL Unreliability.

enum value

enum value

enum value

RHG Roughness (signal roughness experienced).

enum value

RES Restricted (use is restricted to defined sectors).

enum value

OUT Out of tolerance.

enum value

enum value
