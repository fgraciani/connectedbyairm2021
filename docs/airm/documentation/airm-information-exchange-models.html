AIRM and Information Exchange Models

AIRM is not an Information Exchange Model

The AIRM is not an information exchange model â€“ it says nothing about how to encode information. This need is met by community driven information exchange models that are geared to support the implementation of solutions, possibly including specific technology choices, for that community.

However, to facilitate semantic interoperability, the definitions used in information exchange models and in the AIRM are aligned.

The following sections give basic information on key information exchange models used in ATM.

FIXM

The Flight Information Exchange Model (FIXM) is an exchange model capturing Flight and Flow information that is globally standardised.

FIXM website

AIXM

The Aeronautical Information Exchange Model (AIXM) is an exchange model capturing Aeronautical information. Its objective is to enable the provision in digital format of the aeronautical information that is in the scope of Aeronautical Information Services (AIS).

AIXM website

IWXXM

The ICAO Meteorological Information Exchange Model (IWXXM) is an exchange model capturing Meteorological information. It was introduced to support the exchange of meteorological information in accordance with ICAO Annex 3.

IWXXM website

AMXM

The Aerodrome Mapping Exchange Model (AMXM) is an open technical resource owned by EUROCAE / RTCA. The AMXM is provided in support of Aerodrome Mapping Database (AMDB) data exchange. 

AMXM website
