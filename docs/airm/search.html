How does it work?

The AIRM Search provides online access to the latest version of the AIRM, the ATM reference vocabulary. The AIRM Search provides access to thousands of terms, definitions and abbreviations from ICAO documents (annexes, manuals, etc.) and more.

Find more about the AIRM in the AIRM Docs.
